## VACS — Vehicle Access Control System v1.0

## نحوه اجرا

۱. فایل `VACS.html` را دانلود کنید
۲. روی فایل دابل‌کلیک کنید
۳. در مرورگر باز می‌شود
۴. هیچ نصبی لازم نیست

## صفحات سیستم

| صفحه | عملکرد |
|------|--------|
| Login | احراز هویت |
| Dashboard | آمار کلی و رویدادهای اخیر |
| Owners | مدیریت مالکان و پرسنل |
| Vehicles | ثبت و مدیریت خودروها |
| Barriers | ثبت و مدیریت راهبندها |
| Cameras | مدیریت دوربین‌های پلاک‌خوان |
| Access Permissions | اعطا و لغو مجوز دسترسی |
| Access Control | شبیه‌سازی تشخیص پلاک |
| Access Logs | لاگ تردد با فیلتر |
| Reports | آمار و Export CSV |

## مشخصات فنی

| مشخصه | مقدار |
|--------|-------|
| نوع | Single-Page Web Application |
| زبان | HTML5 + CSS3 + JavaScript |
| پایگاه داده | In-Memory JavaScript |
| معماری | 3-Tier + MVC |
| سازگاری | Chrome, Firefox, Safari, Edge |
